# Joke Rating APP

[JRA](https://joke-rating-app.netlify.app/): Vue.js app created from scratch through [Vite](https://vitejs.dev/) that consume [Joke API](https://github.com/15Dkatz/official_joke_api/).

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)
